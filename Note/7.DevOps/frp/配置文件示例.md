# FRP 配置文件示例

## 一、服务端配置示例

### 1.1 基础配置

**frps.toml**:
```toml
# 基础配置
bindAddr = "0.0.0.0"
bindPort = 7000

# 仪表板配置
webServer.addr = "0.0.0.0"
webServer.port = 7500
webServer.user = "admin"
webServer.password = "admin123"

# 日志配置
log.to = "./logs/frps.log"
log.level = "info"
log.maxDays = 3

# 认证配置
auth.method = "token"
auth.token = "your-secure-token-here"
```

### 1.2 完整配置

**frps.toml**:
```toml
# 基础配置
bindAddr = "0.0.0.0"
bindPort = 7000

# KCP 协议（可选，提升性能）
kcpBindPort = 7000

# 仪表板配置
webServer.addr = "0.0.0.0"
webServer.port = 7500
webServer.user = "admin"
webServer.password = "admin123"

# 日志配置
log.to = "./logs/frps.log"
log.level = "info"
log.maxDays = 7

# 认证配置
auth.method = "token"
auth.token = "your-secure-token-here"

# TLS 配置（可选）
transport.tls.forceCert = true
transport.tls.certFile = "/path/to/server.crt"
transport.tls.keyFile = "/path/to/server.key"

# 性能优化
transport.maxPoolCount = 100
transport.tcpMux = true
transport.tcpMuxKeepaliveInterval = 60

# 虚拟主机配置（HTTP/HTTPS）
vhostHTTPPort = 80
vhostHTTPSPort = 443

# 子域名配置
subDomainHost = "yourdomain.com"
```

## 三、特殊场景配置

### 3.1 安全 TCP 代理（STCP）

**服务端配置**:
```toml
# frps.toml
bindAddr = "0.0.0.0"
bindPort = 7000
auth.method = "token"
auth.token = "your-secure-token-here"
```

**客户端配置（服务提供者）**:
```toml
# frpc.toml
serverAddr = "your-server-ip"
serverPort = 7000
auth.method = "token"
auth.token = "your-secure-token-here"

[[proxies]]
name = "secret_ssh"
type = "stcp"
secretKey = "your-secret-key-here"
localIP = "127.0.0.1"
localPort = 22
```

**访问者配置**:
```toml
# frpc_visitor.toml
serverAddr = "your-server-ip"
serverPort = 7000
auth.method = "token"
auth.token = "your-secure-token-here"

[[visitors]]
name = "secret_ssh_visitor"
type = "stcp"
secretKey = "your-secret-key-here"
serverAddr = "your-server-ip"
serverPort = 7000
bindAddr = "127.0.0.1"
bindPort = 6000
```

### 3.2 HTTP Basic Auth

**frpc.toml**:
```toml
[[proxies]]
name = "web"
type = "http"
localIP = "127.0.0.1"
localPort = 8080
customDomains = ["web.yourdomain.com"]

[proxies.plugin]
type = "http_basic_auth"
httpBasicAuth.user = "admin"
httpBasicAuth.password = "admin123"
```

### 3.3 HTTPS2HTTP 转换

**frpc.toml**:
```toml
[[proxies]]
name = "web-https"
type = "https"
localIP = "127.0.0.1"
localPort = 80
customDomains = ["www.yourdomain.com"]

[proxies.plugin]
type = "https2http"
https2http.localAddr = "127.0.0.1:80"
https2http.crtPath = "./client.crt"
https2http.keyPath = "./client.key"
https2http.hostHeaderRewrite = "127.0.0.1"
```

### 3.4 负载均衡

**frpc.toml**:

```toml
customDomains = ["web.yourdomain.com"]
loadBalancer.group = "web"
loadBalancer.groupKey = "web_key"
```

## 五、安全配置

### 5.1 TLS 加密

**服务端配置**:
```toml
# frps.toml
transport.tls.forceCert = true
transport.tls.certFile = "/path/to/server.crt"
transport.tls.keyFile = "/path/to/server.key"
```

**客户端配置**:
```toml
# frpc.toml
transport.tls.enable = true
transport.tls.certFile = "/path/to/client.crt"
transport.tls.keyFile = "/path/to/client.key"
```

### 5.2 IP 白名单

**frps.toml**:
```toml
# 允许的客户端 IP
auth.additionalScopes = ["IP"]

# IP 白名单（需要自定义插件）
# 或使用防火墙规则
```

### 5.3 带宽限制

**frpc.toml**:
```toml
[[proxies]]
name = "web"
type = "http"
localIP = "127.0.0.1"
localPort = 8080
customDomains = ["web.yourdomain.com"]

# 限制带宽
transport.bandwidthLimit = "10MB"
```

## 六、配置文件模板

### 6.1 开发环境模板

**frpc.toml**:
```toml
# 开发环境配置
serverAddr = "dev.yourdomain.com"
serverPort = 7000

# 详细日志
log.to = "./logs/frpc.log"
log.level = "debug"
log.maxDays = 3

# 认证
auth.method = "token"
auth.token = "dev-token"

# 本地开发服务
[[proxies]]
name = "dev-web"
type = "http"
localIP = "127.0.0.1"
localPort = 3000
customDomains = ["dev.yourdomain.com"]
```

### 6.2 生产环境模板

**frps.toml**:
```toml
# 生产环境配置
bindAddr = "0.0.0.0"
bindPort = 7000

# 仪表板（限制访问）
webServer.addr = "127.0.0.1"
webServer.port = 7500
webServer.user = "admin"
webServer.password = "strong-password"

# 日志
log.to = "/var/log/frp/frps.log"

# 强认证
auth.method = "token"

# TLS

transport.tls.certFile = "/etc/ssl/certs/server.crt"
transport.tls.keyFile = "/etc/ssl/private/server.key"

# 性能优化

```
