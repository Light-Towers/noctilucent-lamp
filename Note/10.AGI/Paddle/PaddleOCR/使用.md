  --use_doc_orientation_classify False \

### 2.3 PP-StructureV3 推理

PP-StructureV3 用于文档结构分析,支持表格识别、版面分析等功能。

#### 基本用法
```bash
# 图片URL
paddleocr pp_structurev3 -i https://paddle-model-ecology.bj.bcebos.com/paddlex/imgs/demo_image/pp_structure_v3_demo.png \
  --use_doc_orientation_classify False \
  --use_doc_unwarping False

# 本地图片
paddleocr pp_structurev3 -i ./pp_structure_v3_demo.png

# PDF文件
paddleocr pp_structurev3 -i "2024-5-22 1000家 2024中国·上海国际紧固件工业博览会.pdf" \
  --use_doc_orientation_classify True \
  --use_doc_unwarping True \
  --use_textline_orientation True \
  --save_path ./output
```

#### 参数说明
```bash
# 使用文档方向分类模型
--use_doc_orientation_classify True

# 使用文本图像矫正模块
--use_doc_unwarping True

# 使用文本行方向分类模型
--use_textline_orientation False

# 指定GPU推理
--device gpu
```

### 2.4 PP-ChatOCRv4 推理

PP-ChatOCRv4 结合大模型进行文档理解,需要千帆API Key。

#### 基本用法
```bash
# 需要先获得千帆API Key
paddleocr pp_chatocrv4_doc -i https://paddle-model-ecology.bj.bcebos.com/paddlex/imgs/demo_image/vehicle_certificate-1.png \
  -k 驾驶室准乘人数 \
  --qianfan_api_key your_api_key \
  --use_doc_orientation_classify False \
  --use_doc_unwarping False
```

#### 参数说明
- `-k`: 提取的关键字段
- `--qianfan_api_key`: 百度千帆大模型API Key

## 三、常用参数说明

### 3.1 通用参数
- `--use_doc_orientation_classify`: 是否使用文档方向分类模型
- `--use_doc_unwarping`: 是否使用文本图像矫正模块
- `--use_textline_orientation`: 是否使用文本行方向分类模型
- `--device`: 指定推理设备 (gpu/cpu)
- `--save_path`: 指定结果保存路径

### 3.2 使用建议
1. **文档方向不确定**: 设置 `--use_doc_orientation_classify True`
2. **文档有扭曲变形**: 设置 `--use_doc_unwarping True`
3. **文本行方向不确定**: 设置 `--use_textline_orientation True`
4. **处理PDF文件**: 建议指定 `--save_path` 保存结果

## 四、应用场景

### 4.1 通用OCR识别
适用于: 名片、证件、文档等常规文字识别
```bash
paddleocr ocr -i image.jpg \
  --use_doc_orientation_classify False \
  --use_doc_unwarping False \
  --use_textline_orientation False
```

### 4.2 文档结构分析
适用于: 表格识别、版面分析、文档结构化
```bash
paddleocr pp_structurev3 -i document.pdf \
  --use_doc_orientation_classify True \
  --use_doc_unwarping True \
  --save_path ./output
```

### 4.3 智能文档理解
适用于: 证件信息提取、表单字段识别
```bash
paddleocr pp_chatocrv4_doc -i certificate.png \
  -k "姓名,身份证号,有效期" \
  --qianfan_api_key your_api_key
```
