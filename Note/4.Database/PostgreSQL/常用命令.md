# PostgreSQL 常用命令

## 数据库备份

```bash
pg_dump \
    --no-owner \
    --no-privileges \
    --clean \
    -U root -d dify > dify_backup-$(date +%s).sql
```

**参数说明**:
- `--no-owner`: 不导出对象所有者信息
- `--no-privileges`: 不导出权限信息
- `--clean`: 在创建对象前先删除（清理模式）
- `-U root`: 使用 root 用户
- `-d dify`: 指定数据库名
- `$(date +%s)`: 时间戳文件名

---

## 数据库用户创建

```sql
CREATE USER postgres WITH SUPERUSER PASSWORD 'difyai123456';
```

---

## 数据库恢复

```bash
# 恢复主数据库
psql -U root dify < backup.sql

# 恢复插件数据库
psql -U root dify_plugin < dify_plugin_backup.sql
```

---

## Docker 容器操作

```bash
# 从容器内备份
docker exec pgvector pg_dump --no-owner --no-privileges --clean -U root -d dify > backup.sql

# 导入到容器
docker exec -i pgvector psql -U root dify < backup.sql
```
