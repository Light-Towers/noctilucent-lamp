# 删除idea

* Windows 

参考 https://www.quanxiaoha.com/idea/uninstall-idea.html

* Mac
    ```shell
    rm -rf ~/Library/Preferences/jetbrains.*
    rm -rf ~/Library/Preferences/com.jetbrains.*
    rm -rf ~/Library/Caches/JetBrains
    rm -rf ~/Library/Application\ Support/JetBrains
    ```
