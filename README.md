## 📚 项目目录结构

### 🧪 Code - 项目代码样例
各技术栈的实践代码示例，详见各demo的`README.md`
* `algorithm/` - 算法实现
* `es-demo/` - Elasticsearch示例
* `flink-demo/` - Flink流处理示例
* `hbase-demo/` - HBase示例
* `hive-demo/` - Hive示例
* `java9-ReactiveStream-demo/` - Java响应式流
* `kafka-demo/` - Kafka消息队列
* `my-spring-cloud/` - Spring Cloud微服务
* `nio-demo/` - NIO网络编程

### 💼 Interview - 面试题整理
各类技术岗位面试题目与解答

### 📒 Note - 个人技术知识库
按技术领域分类整理的学习笔记和参考文档

**核心分类：**
1. **算法与数据结构** (`1.Algorithm/`) - 排序、树结构、哈希等算法
2. **大数据技术栈** (`2.Bigdata/`) - Hadoop、Flink、Spark、Kafka等
3. **Java开发** (`3.Java/`) - JVM、并发、集合、Spring等
4. **数据库系统** (`4.Database/`) - MySQL、Redis、MongoDB等
5. **搜索引擎** (`5.Elastic/`) - Elasticsearch原理与使用
6. **网络协议** (`6.Network/`) - HTTP、TCP/IP等网络知识
7. **开发运维** (`7.DevOps/`) - Docker、Linux、Nginx、CI/CD
8. **操作系统** (`8.OperatingSystem/`) - Linux、Windows系统管理
9. **软件工具** (`9.Software/`) - Git、IDE、浏览器工具等
10. **人工智能** (`10.AGI/`) - 大模型、AI工具、神经网络

**编程语言专题：**
* `90.Python/` - Python语法、数据类型、应用案例
* `91.JavaScript/` - 前端与Node.js开发
* `92.Scala/` - Scala语言与函数式编程

**考试认证：**
* `98.软考/` - 系统架构设计师等软考资料

**其他内容：**
* `极客时间/` - 课程学习笔记
* 根目录文件 - 基础参考资料（英语词汇、证书笔记等）

### 🎒 Other - 其他资料
项目相关文档、培训材料、个人技能总结等
