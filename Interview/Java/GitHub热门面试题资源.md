## 四、面试准备建议

### 1. 简历准备
- 突出项目经验和技术栈
- 量化项目成果（性能提升xx%，处理xx万QPS等）
- 准备2-3个有深度的项目讲解

### 2. 算法准备
- LeetCode刷题：先刷热门题（Top 100）
- 重点：数组、链表、树、动态规划、回溯
- 每天保持2-3道题的练习

### 3. 项目准备
- 准备项目架构图
- 技术选型的原因
- 遇到的难点和解决方案
- 性能优化经验

### 4. 面试技巧
- 自我介绍：1-3分钟，突出亮点
- 回答问题：先说结论，再展开细节
- 不会的问题：诚实回答，展示思考过程
- 反问环节：准备有深度的问题

- 重视项目深度和广度
